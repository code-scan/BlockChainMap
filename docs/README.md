# 介绍
[**项目 github 地址**](https://github.com/hepeng10/roc-charts)

此项目基于 zrender 开发，用于展示节点之间的关系。关系的呈现（图谱布局）主要还是由使用者决定，可以很简单的将自己的布局算法加入到 roc-charts 中，而 roc-charts 更多的作用是实现图谱操作功。roc-charts 内置了一些常用功能，也提供了插件机制可供开发更多功能。
